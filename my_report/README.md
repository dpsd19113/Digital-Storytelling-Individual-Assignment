Αρχικά δημιούργησα νέο project στο Unity. Επειτά εφτιξα ένα plain και αλλαξα το lighting ώστε να είναι πράσινο και μετά έβαλα δύο μαύρα plains για να κάνω τους δύο δρόμους.
